this is text about projects and stuff. Here are some of my bigger projects that I have worked on.

Idle Fisher

this is my biggest project that have been working on for a few months. It is an idle game where the player can fish and upgrade to collect more fish. The game is made using GLFW, GLM, and GLEW.

Sunset Maple Drafts

step-sister has a horse business and needed a website. I made one that includes stuff like a mySQL server to store and request reservation dates. It also has a whole admin page to add, remove, update, and search for customers. Also includes a calendar for users to find unreserved dates.

Multiplayer ENET

used ENET for making a multiplayer game. a fun project to work with mulitplayer, send, recieving infromation from a server.

CALCulator

Food App

api stuff, worked with swift to program on IPhones

Card Game
