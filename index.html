Idle Fisher

This is my biggest project called, Idle fisher. This has been a fun project I have been working on for a few months now. It's an idle game where the premise of the game is to fish, upgrade, and idle.

Solo

I am a solo dev. I have done all of the programming. I used to have someone help me with the art, but now I am also making the art by myself too.

Why

When first beginning this project, I wanted a challenge. I've made lots of other unfinished projects previously using game engines, but this time I wanted more of a challenge. I wanted not only to make a game without an engine, but also finish it. So I chose to use a language I was used to, C++, and a library for games like this, SDL2. Right now I am slowly trying to transition my game to use Glad, GLFW, and GLM, for more advanced features like shadows.

Challenges

One of my biggest challenges that I had to overcome was learning all the complex math. Adding Features like collision detection, path finding, shadow, etc. I love math, and trying to find solutions and explanations to problems. It took lots of time and research to get more comfortable with these complex features.
